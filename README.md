## Run Locally

Clone the project

```bash
  git clone https://github.com/edermarcos/angular_geekcode-client.git geekcode-client
```

Go to the project directory

```bash
  cd geekcode-client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
